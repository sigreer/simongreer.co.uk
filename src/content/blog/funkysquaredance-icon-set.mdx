## Background

Inspired by Alecive's Flatwoken icon set, this is (the start of) a new set of icons with a chunkier, more button-like aesthetic. I will slowly add more icons to this pack as I work my way through the apps I use.

## The apps

- **[Astro](https://astro.build)** The web framework used to build this website.
- **Google Calendar** One of the few bits of degoogling I've not been able to complete.
- **[Hoppscotch](https://github.com/hoppscotch/hoppscotch)** An open source tool for interacting with APIs (like Postman but better and cost free)
- **[BlueSky](https://bsky.app)** Social app that I suspect you'll have heard of by now
- **[Cursor](https://cursor.sh)** My IDE of choice. It's VScode with robot brains.
- **[Follow.is](https://follow.is)** An open source, nextgen feed reader. Social media, news, Youtube, RSS + more housed in a slick UI.
- **[Anthropic](https://anthropic.com)** The creators of Claude, which is my LLM of choice currently.
- **[Suyu](https://suyu.dev)** A Switch emulator that I just found out is now defunct. Annoying as I spent a long time on this icon!
